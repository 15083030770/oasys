# oa_system
办公自动化（OA）是面向组织的日常运作和管理，员工及管理者使用频率最高的应用系统，自1985年国内召开第一次办公自动化规划会议以来，OA在应用内容的深度与广度、IT技术运用等方面都有了新的变化和发展，并成为组织不可缺的核心应用系统。主要推行一种无纸化办公模式。
